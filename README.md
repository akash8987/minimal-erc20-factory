# Minimal ERC-20 Token Factory

This repository provides a streamlined, secure, and gas-optimized implementation of the ERC-20 standard using OpenZeppelin's industry-leading libraries. It is designed for developers who need to deploy standard fungible tokens with minimal overhead.

## Features
- **Standard Compliance**: Fully compatible with the ERC-20 interface.
- **Security**: Built on top of OpenZeppelin's battle-tested contracts.
- **Ease of Use**: Single-file deployment structure for rapid prototyping.

## Quick Start
1. Install dependencies: `npm install`
2. Compile the contract: `npx hardhat compile`
3. Deploy to your network of choice using the provided script.
